# Julius Kwaku Antwi — Portfolio

A clean, responsive, one-page portfolio website built with **HTML, CSS, and JavaScript**.

## What to edit
Open `index.html` and update:
- Project links (Live Demo / GitHub Repo)
- Any text you want

## Run locally
Double-click `index.html` or use VS Code Live Server.

## Deploy on GitHub Pages
1. Push this folder to a GitHub repository
2. Repo **Settings → Pages**
3. Source: Deploy from branch → `main` and `/ (root)`
4. Your site will be live at: `https://YOUR-USERNAME.github.io/YOUR-REPO/`
